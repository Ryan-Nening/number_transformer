# Number Transformer

## Description
This program is an Object-Oriented Graphical User Interface (GUI) tool built with Python and `tkinter`. It is designed to act as a mathematical data cruncher. The program reads a text file containing a list of integers, calculates the square and cube of each number, and automatically generates two separate output files (`squares.txt` and `cubes.txt`). It also features a real-time preview dashboard and built-in error handling.

## Features
* **Modern Dark Theme:** A sleek, tech-focused interface featuring custom fonts and a flat-design interactive button.

## How to Use
* **Prepare the Data:** Create a text file named numbers.txt in the same directory as the scripts. Add a list of whole numbers to this file, placing one integer on each line (e.g., 2, 5, 10).
* **Run the Program:** Open your terminal or IDE (like VS Code) and execute the main driver file:

  ``python main_program_for_transformer.py``

* **Transform Data:** When the dashboard opens, click the "Select File & Transform" button.
* **Select the File:** Use the file explorer to locate and select your numbers.txt file.
* **View Results:** Watch the live preview box populate with your calculations. Check your project folder to find your newly generated squares.txt and cubes.txt files.
